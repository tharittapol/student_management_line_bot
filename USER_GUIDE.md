บอทเชื่อมต่อกับ Google Sheet ตารางเรียนโดยตรง ทุกคำสั่งต้องขึ้นต้นด้วย `/` และใช้งานใน LINE group ที่ได้รับอนุญาตเท่านั้น

## 1. ดูตารางเรียนสัปดาห์นี้

แสดงตารางเรียนจากแท็บ `สัปดาห์นี้` ใน Google Sheet จัดกลุ่มตามวัน เรียงตามเวลา พร้อมเบอร์โทรผู้ปกครอง

**คำสั่ง**

```
/ตารางเรียน
```

**ตัวอย่างคำตอบของบอท**

```
📚 ตารางเรียนในแท็บสัปดาห์นี้
12-16 พ.ค. 2569

📅 วันจันทร์ 12 พ.ค. 2569
1. ⏳ แพรว (แพรวา ศิริพงษ์) | English Foundation
จ. 12 พ.ค. 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม. | 📞 081-234-5678

2. ✅ บอส (ธนากร ใจดี) | คณิตศาสตร์ ม.2
จ. 12 พ.ค. 15:00-17:00 | ชม.5,6 | เหลือ 20 ชม.

📅 วันพุธ 14 พ.ค. 2569
3. 🏖️ มิน (ณัฐธิดา วงศ์วาน) | IELTS Speaking
พ. 14 พ.ค. 10:00-11:00 | ชม.2 | ลา | เหลือ 11 ชม. | 📞 089-876-5432

4. ✅ ต้น (กฤติน ภักดี) | Physics ม.4
พ. 14 พ.ค. 13:00-15:00 | ชม.11,12 | เหลือ 20 ชม.
```

**อ่านผลลัพธ์**

| ส่วน | ความหมาย |
|------|----------|
| `⏳ / ✅ / 🏖️` | สถานะ: รอคอนเฟิร์ม / คอนเฟิร์มแล้ว / ลา |
| `แพรว (แพรวา ศิริพงษ์)` | ชื่อเล่น (ชื่อเต็ม) |
| `English Foundation` | คอร์สที่เรียน |
| `จ. 12 พ.ค. 13:00-15:00` | วันและเวลาเรียน |
| `ชม.7,8` | ชั่วโมงที่ 7 และ 8 ของคอร์ส |
| `เหลือ 14 ชม.` | ชั่วโมงเรียนที่เหลือ |
| `📞 081-234-5678` | เบอร์โทรผู้ปกครอง |

> บอทจะส่งตารางนี้ให้อัตโนมัติทุกวัน 09:00

## 2. ดูตารางเวลา slot ว่าง/จอง

แสดงว่าแต่ละชั่วโมงในช่วง 10:00–17:00 มีนักเรียนคนไหนเรียนอยู่ หรือว่างสำหรับจัดตาราง
นักเรียนที่ลาจะไม่แสดงในตาราง

**คำสั่ง**

```
/ตารางเวลา
```

**ตัวอย่างคำตอบของบอท**

```
⏰ ตารางเวลาสัปดาห์นี้

📅 วันจันทร์ 12 พ.ค. 2569
10-11  ว่าง
11-12  ว่าง
12-13  ว่าง
13-14  แพรว
14-15  แพรว, บอส
15-16  บอส
16-17  ว่าง

📅 วันพุธ 14 พ.ค. 2569
10-11  ว่าง
11-12  ว่าง
12-13  ว่าง
13-14  ต้น
14-15  ต้น
15-16  ว่าง
16-17  ว่าง
```

> มิน (ลา) ไม่แสดงในตารางเวลา แม้จะมีคาบเรียนวันพุธ

> บอทจะส่งตารางนี้พร้อมกับ `/ตารางเรียน` ทุกวัน 09:00

## 3. ดูข้อมูลนักเรียน

แสดงรายชื่อนักเรียนที่ยังเรียนไม่จบทั้งหมด พร้อมชื่อผู้ปกครองและเบอร์โทร
ใช้ดูชื่อเล่น-ชื่อจริงสำหรับพิมพ์คำสั่งอื่นๆ

**คำสั่ง**

```
/ข้อมูลนักเรียน
```

**ตัวอย่างคำตอบของบอท**

```
👥 นักเรียนที่ยังเรียนไม่จบ (4 คน)
1. แพรว - แพรวา | English Foundation
   👨‍👩‍👧 สมชาย ศิริพงษ์ | 📞 081-234-5678
2. บอส - ธนากร | คณิตศาสตร์ ม.2
3. มิน - ณัฐธิดา | IELTS Speaking
   👨‍👩‍👧 สมหญิง วงศ์วาน | 📞 089-876-5432
4. ต้น - กฤติน | Physics ม.4, Chemistry ม.5
   👨‍👩‍👧 ประสิทธิ์ ภักดี | 📞 062-111-2222
```

> ถ้านักเรียน 1 คนเรียนหลายคอร์ส จะแสดงรวมในบรรทัดเดียว คั่นด้วย `,`

## 4. อัพเดทเวลาเรียน

เปลี่ยนวันและเวลาเรียนของนักเรียน บอทจะบันทึกลง Google Sheet ทันที

**รูปแบบคำสั่ง**

```
/อัพเดท ชื่อเล่น ชื่อจริง วัน/เดือน เวลาเริ่ม-เวลาจบ
```

**ตัวอย่าง — เปลี่ยนเป็นวันที่ 9 พ.ค. เวลา 13:00-15:00**

```
/อัพเดท แพรว แพรวา 9/5 13:00-15:00
```

คำตอบของบอท:

```
🔄 อัพเดทเวลาเรียน
⏳ แพรว (แพรวา ศิริพงษ์) | English Foundation
ศ. 9 พ.ค. 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม. | 📞 081-234-5678
```

**ตัวอย่าง — ระบุปี พ.ศ.**

```
/อัพเดท แพรว แพรวา 9/5/2570 13:00-15:00
```

คำตอบของบอท:

```
🔄 อัพเดทเวลาเรียน
⏳ แพรว (แพรวา ศิริพงษ์) | English Foundation
ศ. 9 พ.ค. 2570 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม. | 📞 081-234-5678
```

> หลังอัพเดท สถานะจะถูกรีเซ็ตเป็น ⏳ รอคอนเฟิร์มใหม่เสมอ

## 5. คอนเฟิร์มเวลาเรียน

ยืนยันเวลาเรียนของนักเรียน รองรับ 3 รูปแบบ

---

### 5.1 คอนเฟิร์มด้วยชื่อ

**รูปแบบคำสั่ง**

```
/คอนเฟิร์ม ชื่อเล่น ชื่อจริง
```

**ตัวอย่าง**

```
/คอนเฟิร์ม แพรว แพรวา
```

คำตอบของบอท:

```
✅ คอนเฟิร์มเวลาเรียน
✅ แพรว (แพรวา ศิริพงษ์) | English Foundation
จ. 12 พ.ค. 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม. | 📞 081-234-5678
```

---

### 5.2 คอนเฟิร์มพร้อมเปลี่ยนเวลา

**รูปแบบคำสั่ง**

```
/คอนเฟิร์ม ชื่อเล่น ชื่อจริง วัน/เดือน เวลาเริ่ม-เวลาจบ
```

**ตัวอย่าง**

```
/คอนเฟิร์ม แพรว แพรวา 9/5 14:00-16:00
```

คำตอบของบอท:

```
✅ คอนเฟิร์มเวลาเรียน
✅ แพรว (แพรวา ศิริพงษ์) | English Foundation
ศ. 9 พ.ค. 14:00-16:00 | ชม.7,8 | เหลือ 14 ชม. | 📞 081-234-5678
```

---

### 5.3 คอนเฟิร์มหลายคนพร้อมกันด้วยเลขลำดับ

ใช้เลขลำดับจาก `/ตารางเรียน` คั่นด้วย `,`

**รูปแบบคำสั่ง**

```
/คอนเฟิร์ม เลข,เลข,เลข
```

**ตัวอย่าง — คอนเฟิร์มลำดับที่ 1, 3 และ 4**

```
/คอนเฟิร์ม 1,3,4
```

คำตอบของบอท:

```
✅ คอนเฟิร์มเวลาเรียน

1. ✅ แพรว (แพรวา ศิริพงษ์) | English Foundation
จ. 12 พ.ค. 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม.

3. ✅ มิน (ณัฐธิดา วงศ์วาน) | IELTS Speaking
พ. 14 พ.ค. 10:00-11:00 | ชม.2 | เหลือ 11 ชม.

4. ✅ ต้น (กฤติน ภักดี) | Physics ม.4
พ. 14 พ.ค. 13:00-15:00 | ชม.11,12 | เหลือ 20 ชม.
```

**ตัวอย่าง — ใส่เลขที่ไม่มีในตาราง**

```
/คอนเฟิร์ม 1,99
```

คำตอบของบอท:

```
✅ คอนเฟิร์มเวลาเรียน

1. ✅ แพรว (แพรวา ศิริพงษ์) | English Foundation
จ. 12 พ.ค. 13:00-15:00 | ชม.7,8 | เหลือ 14 ชม.

⚠️ ไม่พบลำดับที่ 99
```

## 6. ไม่คอนเฟิร์มเวลาเรียน

ยกเลิกการยืนยัน เปลี่ยนสถานะกลับเป็น ⏳ รองรับทั้งระบุชื่อและเลขลำดับ เช่นเดียวกับ `/คอนเฟิร์ม`

---

### 6.1 ไม่คอนเฟิร์มด้วยชื่อ

```
/ไม่คอนเฟิร์ม บอส ธนากร
```

คำตอบของบอท:

```
⏳ ไม่คอนเฟิร์มเวลาเรียน
⏳ บอส (ธนากร ใจดี) | คณิตศาสตร์ ม.2
จ. 12 พ.ค. 15:00-17:00 | ชม.5,6 | เหลือ 20 ชม.
```

---

### 6.2 ไม่คอนเฟิร์มหลายคนด้วยเลขลำดับ

```
/ไม่คอนเฟิร์ม 2,4
```

คำตอบของบอท:

```
⏳ ไม่คอนเฟิร์มเวลาเรียน

2. ⏳ บอส (ธนากร ใจดี) | คณิตศาสตร์ ม.2
จ. 12 พ.ค. 15:00-17:00 | ชม.5,6 | เหลือ 20 ชม.

4. ⏳ ต้น (กฤติน ภักดี) | Physics ม.4
พ. 14 พ.ค. 13:00-15:00 | ชม.11,12 | เหลือ 20 ชม.
```

## 7. บันทึกการลา

บันทึกสถานะ "ลา" ให้นักเรียน บอทจะแสดงอีโมจิ 🏖️ แทน ⏳/✅ และจะไม่แสดงในตารางเวลา

**รูปแบบคำสั่ง**

```
/ลา ชื่อเล่น ชื่อจริง
```

**ตัวอย่าง**

```
/ลา มิน ณัฐธิดา
```

คำตอบของบอท:

```
📝 อัพเดทสถานะการเรียน: ลา
🏖️ มิน (ณัฐธิดา วงศ์วาน) | IELTS Speaking
พ. 14 พ.ค. 10:00-11:00 | ชม.2 | ลา | เหลือ 11 ชม. | 📞 089-876-5432
```

**รูปแบบคำสั่ง**

```
/เข้าเรียน ชื่อเล่น ชื่อจริง
```

**ตัวอย่าง**

```
/เข้าเรียน มิน ณัฐธิดา
```

คำตอบของบอท:

```
📝 อัพเดทสถานะการเรียน: เข้าเรียนปกติ
⏳ มิน (ณัฐธิดา วงศ์วาน) | IELTS Speaking
พ. 14 พ.ค. 10:00-11:00 | ชม.2 | เข้าเรียนปกติ | เหลือ 11 ชม. | 📞 089-876-5432
```

## 9. ดู LINE Group ID

แสดง ID ของ LINE group นี้ สำหรับนำไปเพิ่มในการตั้งค่าบอท
คำสั่งนี้ใช้ได้ในทุก group แม้ยังไม่ได้เพิ่มในระบบ

**คำสั่ง**

```
/groupid
```

**ตัวอย่างคำตอบของบอท**

```
🆔 LINE group id
C1a2b3c4d5e6f7a8b9c0d1e2f3a4b5c6
```

## 11. รูปแบบวันที่ที่รองรับ

| รูปแบบ | ตัวอย่าง | ความหมาย |
|--------|---------|----------|
| `วัน/เดือน เวลา` | `9/5 13:00-15:00` | วันที่ 9 พ.ค. ปีปัจจุบัน |
| `วัน/เดือน/ปี พ.ศ. เวลา` | `9/5/2570 13:00-15:00` | วันที่ 9 พ.ค. 2570 (= ค.ศ. 2027) |
| `วัน/เดือน/ปี ค.ศ. เวลา` | `2027-05-09 13:00-15:00` | รูปแบบ ISO |
| `วัน เดือนไทย เวลา` | `9 พ.ค. 13:00-15:00` | ชื่อเดือนย่อภาษาไทย |
| `วัน เดือนไทย ปี พ.ศ. เวลา` | `9 พ.ค. 2570 13:00-15:00` | ระบุปีด้วยชื่อเดือนไทย |

**เดือนไทยย่อที่รองรับ**

ม.ค. / ก.พ. / มี.ค. / เม.ย. / พ.ค. / มิ.ย. / ก.ค. / ส.ค. / ก.ย. / ต.ค. / พ.ย. / ธ.ค.

## 12. ความหมายของอีโมจิ

| อีโมจิ | ความหมาย |
|--------|----------|
| ✅ | คอนเฟิร์มแล้ว |
| ⏳ | รอคอนเฟิร์ม |
| 🏖️ | ลา (ไม่แสดงใน `/ตารางเวลา`) |
| 📞 | เบอร์โทรผู้ปกครอง |
| 👨‍👩‍👧 | ชื่อผู้ปกครอง |
| ⚠️ | แจ้งเตือนข้อผิดพลาด เช่น ไม่พบนักเรียน |

## ข้อควรรู้

- **ชื่อเล่นและชื่อจริง** ไม่แยกตัวพิมพ์ใหญ่-เล็ก และดูได้จาก `/ข้อมูลนักเรียน`
- **เลขลำดับ** ใน `/คอนเฟิร์ม 1,2` ตรงกับตัวเลขที่แสดงใน `/ตารางเรียน` เสมอ
- **ข้อความยาว** บอทจะแบ่งส่งอัตโนมัติ ไม่ตัดกลางประโยค
- **การแจ้งเตือนอัตโนมัติ** บอทส่ง `/ตารางเรียน` และ `/ตารางเวลา` ให้ทุกวัน 09:00 โดยไม่ต้องสั่ง
